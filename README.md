🎯 Aplikasi Rekomendasi Karier Berdasarkan Minat dan Keahlian
A career recommendation CLI application built in Go that helps users discover suitable career paths based on their interests and skills.

✨ Features
- Add, update, and delete personal interests and skills
- Get career recommendations based on inputted competencies
- Search career paths by name or industry category (Sequential & Binary Search)
- Sort recommendations by match percentage or average salary (Selection & Insertion Sort)
- View match percentage statistics for each recommended career path

🛠️ Tech Stack

Language : Go (Golang)
Type : CLI Application
